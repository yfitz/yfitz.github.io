Arrays und Objekte in JavaScript

Kapitel 3

Array ist eine Liste von Elementen.

Zwei Arten dieses zu deklarieren:


	var arrayName = [value0, value1];
	var arrayName = new Array(value0, value1);

Die Element können unterschiedlichen Datentyp haben.

			
	var rainbow = ['Red', 'Orange', 'Yellow', 'Green', 'Blue'];

	var lotteryNumbers = [33, 72, 64, 18, 17, 85];

	var myFavoriteThings = ['Broccoli', 1024, 'Sherlock'];

Länge und Elemente

Die Eigenschaft length zählt die Elemente.


	var rainbow = ['Red', 'Orange', 'Yellow', 'Green','Blue'];
	console.log(rainbow.length);

Zugriff auf Elemente mit [] und Index.(beginnt bei 0)


	var firstColor = rainbow[0];
	var lastColor  = rainbow[4];

Verändern von Array Elementen


	rainbow[3] = "White";

Verlängern von Arrays (Arrays haben keine feste Länge.)


	rainbow.push('Dancing');

Schleifen durch Arrays

for-Schleife 

var rainbow = ['Red', 'Orange', 'Yellow', 'Green','Blue'];

for (var i = 0; i < rainbow.length; i++) {
  console.log(rainbow[i]);
}

foreach-Schleife

array.forEach(function(arrayElement) {
                console.log(arrayElement);
});

Beispiel oben als foreach:

rainbow.forEach(function(arrayElement) {
  console.log(arrayElement);
});

	var points = [40, 100, 1, 5, 25, 10];

	points.sort(function(a, b){
	
		if(a>b) { return 1;}	// b vor a

Objekte

Objekte (Objekttyp ist vordefiniert: 'Object')


	Syntax: 

	var objektName = {
		eigenschaftsName:  eigenschaftsValue,
		eigenschaftsName:  eigenschaftsValue
	};
			

	Beispiel:

	var user = {
		hometown: 'Atlanta, GA',
		hair: 'Auburn',
		likes: ['knitting', 'code'],
		birthday: {month: 10, day: 17}
	};

Zugriff auf Objekte

Mit der Punkt Notation kann man auf das Objekt zugreifen.


	var user = {
		hometown: 'Atlanta, GA',
		hair: 'Auburn'
	};

	var usersHometown = user.hometown;

oder mit der Klammer Notation (so wie bei Arrays)


	var usersHair = user['hair'];

Objekte verändern


	var user = {
		hometown: 'Atlanta, GA',
		hair: 'braun'
	};

	user.hair = 'blond';    	// Eigenschaft veraendern

	user.married = true; 	// Neue Eigenschaften hinzufügen

	delete user.married; 	// oder löschen.

	for(var key in user) {
		// Eigenschaft ausgeben
		document.write( "Eigenschaft:"+ key );  
		
		// Wert ausgeben
		document.write( "Wert:"+ user[key] ); 	
	}

Objekt Methoden


	var jolene = {
		age: 21,
		hairColor: 'Auburn',
		talk: function() {
			console.log('Hello!');
			},
		eat: function(food) {
			console.log('Yum, I love ' + food);
		}
	};

	jolene.talk();

	jolene.eat('pizza');   //Methodenaufruf mit der Punkt-Notation

Arrays von Objekten

Weil Arrays verschiedene Datentypen enthalten können, können diese auch Objekte enthalten.


	var users = [
		{name: 'Jolene', age: 21},
		{name: 'Alexa',  age: 18}
	];

	for (var i = 0; i < users.length; i++) {
		var user = users[i];
		console.log(user.name + ' is ' + user.age + ' years old.');
	}

Objekte

Objekte können auch als Argumente an Funktionen übergeben werden.


	var jolene = {
		age: 21,
		hairColor: 'Auburn',
		likes: ['pizza', 'tacos'],
		birthday: {month: 3, day: 14, year: 1995}
	}

	function describeUser(user) {
		console.log('You are ' + user.age + ' years old with '
				+ user.hairColor + ' hair.');
	}

	describeUser(jolene);

Aufgaben

1.Erzeugen Sie ein Objekt mit folgende Eigenschaften:

  • Titel
  • Verlag
  • Seitenanzahl
  • Autor


2. Legen Sie jetzt ein Array von Büchern an.Erst einlesen und speichern, dann in einer Schleife ausgeben.

3. Legen Sie eine Objekt Methode 'ausgeben' an.

Lösung


	var buch_array = []; var titel; var verlag; var seiten; var autor;
	for (var i = 0; i < 3; i++) {
		titel = prompt("Titel eingeben");
		verlag = prompt("Verlag eingeben");
		seiten = prompt("Seitenanzahl eingeben");
		autor = prompt("Autor eingeben");
		
		buch_array.push({
			Titel: titel,
			Verlag: verlag,
			Seitenanzahl: parseInt(seiten),
			Autor: autor,
			ausgeben: function() {
				document.write("Titel: " + this.Titel +
				"Verlag: " + this.Verlag +
				"Seitenanzahl: " + this.Seitenanzahl +
				"Autor: " + this.Autor );
			}
		});
	}
	for (var i = 0; i < 3; i++) {
		buch_array[i].ausgeben();
	}
