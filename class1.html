Was ist JavaScript?

  • Eine client-seitige Sprache. Der Browser liest den Code und führt in direkt aus.
  • Interfaces mit HTML & CSS lassen uns dynamische Webseiten entwickeln, die auf Benutzereingaben reagieren.

Variable deklarieren (erzeugen) mit var oder let


 var numberOfKittens;				\\ Deklaration
 let numberOfDogs;
 numberOfKittens = 5;				\\ Initialisierung

Was ist der Unterschied zwischen var und let?

			
function varTest() {
	var x = 31;
	if (true) {
		var x = 71;  					
		console.log(x);  			// 71
	}
	console.log(x);  				// 71
}

function letTest() {
	let x = 31;
	if (true) {
		let x = 71;  					
		console.log(x);  			// 71
	}
	console.log(x);  				// 31
}

Gültigkeitsbereich (Scope) von let ist die Funktion und von var ist der Block.

Variablen

  • ... ohne Wert sind undefined.
  • ... können unterschiedliche Datentypen enthalten.
  • ... können sich im Wert ändern.

Variablennamen

  • ... sind case-sensitive.
  • ... beginnen mit einem Buchstaben,$ oder _ .
  • ... enthalten $, Ziffern, Buchstaben oder _.

Vermeide reservierte Wörter.

Aussagekräftige Variablennamen sind wichtig für das Verständnis ihres Codes.

Sechs Datentypen sind primitives:


let userName = 'Jane Lane'; 		// String (Zeichenketten)
let z = 123e-5;     			// Number (Integer und Float)
let catsAreBest = true; 			// Boolean
let favoriteThings; 			// Undefined (kein Wert zugewiesen)
let goodPickupLines = null; 		// Null (leerer Wert)
let id = Symbol("id");			//Symbol Datentyp

Er kann als Schlüssel einer Eigenschaft eines Objekts verwendet werden. 
In manchen Programmiersprachen werden Symbole auch Atoms genannt.

 let kurs = Object();  // ein Objekt von Typ Object

Zeichenketten in Zahlen umwandeln:


let string = "5.3.4abc";
let ganzeZahl = parseInt(string);			// 5
let floatZahl = parseFloat(string);		// 5.3

NaN = Not-A-Number (Kann nicht umgewandelt werden!)

Strings

Variablen können Zeichenketten(strings) enthalten. Die Zeichenketten stehen in einfachen oder doppelten Anführungszeichen.


var hisName = 'Struppy';
var herName = "Mausi";

Ein Anführungszeichen escape mit Backslash


console.log('Ich ein Zeichen\' hier');

Zeichenketten Operatoren

Zeichenketten zusammenfügen mit a +, der concatenation Operator.

var kittensName = 'Fluffy ';
var fullName = kittensName + 'McDougle';
console.log(fullName); // Outputs 'Fluffy McDougle'

An Zeichenkette am Ende anfügen mit +=.

var kittensName = 'Admiral ';
kittensName += 'Snuggles';
console.log(kittensName); // Outputs 'Admiral Snuggles'

Kombiniere Zeichenketten und Zahlen

In Konkatenation von Zeichenketten und Zahlen, werden Zahlen wie Zeichenketten behandelt.

var numberOfFruit = 6;
var typeOfFruit = 'bananas';
var allTheFruit = 'I have ' + numberOfFruit + ' ' + typeOfFruit + '!';
console.log(allTheFruit);
